# ComputerStorage API

The purpose of this repo is to practice unit testing with [Mocha](https://mochajs.org/) and [Chai](https://www.chaijs.com/api/bdd/).

## Usage

Install devDependencies

```shell
npm install
```

Run tests

```shell
npm test
```
